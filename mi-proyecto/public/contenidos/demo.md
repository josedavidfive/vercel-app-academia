# Introducción a la Informática

## ¿Qué es la informática?
La informática es la ciencia que estudia el tratamiento automático de la información mediante sistemas computacionales.

## Conceptos fundamentales

### Hardware
Componentes físicos de un sistema informático: procesador, memoria RAM, disco duro, tarjeta gráfica.

### Software
Conjunto de programas e instrucciones que controlan el hardware: sistemas operativos, aplicaciones, utilidades.

### Redes
Conjunto de dispositivos conectados para compartir información y recursos.

## Seguridad informática
Protección de sistemas y datos frente a accesos no autorizados, ataques y pérdidas de información.

### Principios CIA
- **Confidencialidad**: solo acceden quienes están autorizados
- **Integridad**: los datos no se alteran sin autorización
- **Disponibilidad**: el sistema está accesible cuando se necesita
